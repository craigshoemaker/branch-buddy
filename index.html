Note: Check out the quick reference for help with common tasks.

  1. Fork the azure-docs-pr repository

  2. Clone the fork to your machine:

    git clone https://github.com/<USER_NAME>/azure-docs-pr.git

  3. Add a remote named upstream:

    git remote add upstream http://github.com/MicrosoftDocs/azure-docs-pr.git

  4. Fetch the release branch:

    git fetch upstream <BRANCH_NAME>

  5. Create local branch:

    git checkout -B <BRANCH_NAME> upstream/<BRANCH_NAME>
